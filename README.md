# mksurvey-lab

## 项目简介

mksurvey-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为一系列静态页面的集中存放与展示空间。

这些页面可能包含调研、演示、原型、实验性功能或信息展示等内容，所有页面均为独立 HTML 文件，可直接通过浏览器访问。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名建议体现内容主题。
- `assets/`：存放各页面可能共用的样式、脚本或图片资源，避免重复。
- `index.html`（可选）：可作为仓库的导航或入口页。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，互不影响。
- 如需新增页面，请在 `pages/` 目录下创建新的 HTML 文件，并确保引用资源路径正确。
- 建议每个页面包含基本的 meta 信息（如字符编码、视口设置等），以保证在不同浏览器中的兼容性。
- 本仓库不限制页面内容类型，但请确保内容合法合规。

## 维护说明

- 本仓库欢迎提交改进、修复或新增页面。
- 提交前请检查页面是否能在本地正常打开，资源链接是否有效。
- 请保持目录结构清晰，避免文件混乱。
- 如涉及公共资源更新，请同步更新所有使用该资源的页面引用。
- 本仓库不提供自动部署或持续集成配置，如需部署到服务器，请自行配置。
